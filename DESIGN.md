# 🎨 롯데백화점 탈출하기 — 디자인 문서

## 1. 기획 의도

롯데백화점 고객을 대상으로 한 **참여형 이벤트 게임**입니다.
방탈출이라는 친숙한 게임 포맷에 백화점 각 층의 특성을 녹여,
고객이 매장을 자연스럽게 둘러보는 듯한 경험을 제공합니다.
탈출 성공 시 지점별 할인 쿠폰을 제공하여 실제 방문/구매로 연결되도록 설계했습니다.

## 2. 사용자 흐름 (User Flow)

```
지점 선택 → 난이도 선택 → 퍼즐 5단계 진행 → 성공(쿠폰) / 실패(재도전)
```

1. **지점 선택**: 나의 지점을 고른다 (몰입감 + 쿠폰 개인화)
2. **난이도 선택**: 쉬움/보통/어려움으로 제한 시간 결정
3. **퍼즐 진행**: 층별 테마 퍼즐을 순서대로 해결
4. **결과**: 시간 내 탈출 시 쿠폰, 실패 시 재도전

## 4. 화면 구성

| 화면 | 설명 |
|------|------|
| 지점 선택 | 12개 지점 그리드 버튼 |
| 시작 화면 | 지점 사진 + 영업시간 + 난이도 선택 |
| 게임 화면 | 타이머 · 힌트 · 진행바 · 퍼즐 영역 |
| 성공 화면 | 탈출 시간 + 쿠폰 + 축하 컨페티 |
| 실패 화면 | 도달한 층 표시 + 재도전 |

## 5. 퍼즐 설계

각 퍼즐은 해당 층의 매장 특성을 반영합니다.

| 단계 | 층 | 매장 | 퍼즐 유형 | 정답 메커니즘 |
|------|----|------|-----------|---------------|
| 1 | B1 | 식품관 | 순서 정렬 | 조리 순서대로 클릭 |
| 2 | 1F | 화장품 | 조합 선택 | 지정된 컬러 3개 순서 선택 |
| 3 | 3F | 패션 | 다중 선택 | 비즈니스 캐주얼 아이템 3개 |
| 4 | 5F | 키즈 | 이미지 퀴즈 | 캐릭터 사진+설명 보고 이름 맞추기 |
| 5 | 출구 | 탈출 | 단어 조합 | '롯데백화점' 글자 맞추기 |

### 난이도 배려
- 캐릭터 퀴즈는 사진과 설명을 함께 제공해 처음 보는 고객도 추론 가능
- 각 퍼즐은 오답 시 즉시 "다시 시도" 피드백 제공
- 힌트 3회로 막힘 방지

## 6. 게임 메커니즘

### 타이머
- 난이도별 제한 시간 (600/420/300초)
- 60초 이하 시 빨간색 + 깜빡임 경고
- 시간 초과 시 실패 화면

### 힌트 시스템
- 총 3회 제공
- 각 퍼즐별 맞춤 힌트
- 소진 시 버튼 비활성화

### 쿠폰 발급
- 지점별 차등 혜택 (뷰티/패션/리빙/전 매장 등)
- 고유 쿠폰 코드 자동 생성 (`LOTTE-{지점}-{랜덤}`)

## 7. 반응형 설계

- 최대 너비 480px (모바일 우선)
- 380px 이하 작은 화면 대응 미디어 쿼리
- 터치 친화적 버튼 크기

## 8. 접근성 고려

- 충분한 색 대비 (다크 배경 + 밝은 텍스트)
- 명확한 피드백 (정답/오답 시각적 표시)
- 큰 터치 타깃

## 9. 향후 개선 아이디어

- 랭킹/리더보드 추가
- 퍼즐 종류 다양화 및 랜덤 출제 확대
- 실제 쿠폰 발급 시스템 연동
- 다국어 지원 (외국인 고객 대응)
