# 🏬 롯데백화점 탈출하기

폐점 시간이 지난 롯데백화점에서, 각 층의 퍼즐을 풀고 탈출하는 웹 기반 방탈출 게임입니다.
고객 참여형 이벤트 콘텐츠로 제작되었으며, 탈출에 성공하면 선택한 지점의 할인 쿠폰을 받을 수 있습니다.

## 🎮 플레이

👉 **[게임 플레이하기](https://yeeun0560-star.github.io/lotte-escape/)**

## 📖 게임 소개

> "폐점 시간이 지났습니다! 5개 층의 퍼즐을 풀고 백화점을 탈출하세요!"

지점을 선택하고 난이도를 고른 뒤, 백화점 각 층을 돌아다니며 퍼즐을 풉니다.
제한 시간 안에 모든 층을 통과하면 탈출 성공!

## 🕹️ 게임 구성

### 1. 지점 선택
전국 12개 주요 롯데백화점 지점 중 하나를 선택합니다.
(잠실 · 본점 · 강남 · 부산본점 · 대전 · 광복 · 센텀시티 · 대구 · 광주 · 수원 · 인천터미널 · 동탄)

### 2. 난이도 선택
- 🟢 쉬움 (10분)
- 🟡 보통 (7분)
- 🔴 어려움 (5분)

### 3. 퍼즐 5단계
| 층 | 매장 | 퍼즐 내용 |
|----|------|-----------|
| B1 | 식품관 | 요리 레시피 순서 맞추기 |
| 1F | 화장품 | 시그니처 컬러 조합 |
| 3F | 패션 | 비즈니스 캐주얼 코디 완성 |
| 5F | 키즈 | 킨더유니버스 캐릭터 맞추기 |
| 출구 | 탈출 | 최종 단어 맞추기 |

### 4. 보상
탈출 성공 시 **선택한 지점의 할인 쿠폰**을 발급받습니다.

## ✨ 주요 기능
